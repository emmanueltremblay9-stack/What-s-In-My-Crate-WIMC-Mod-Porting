Supported Crates
Industrial storage targets
<&crate_recipe>Wooden Storage Crates are supported when Immersive Engineering is loaded.
The preview reads stored item data from the stack and shows a 9-wide grid when enough screen space is available.<br>
If a crate is empty, malformed, or has not generated loot yet, the tooltip reports that state instead of crashing.<np>
<&reinforced_crate_recipe>Reinforced Storage Crates use the same preview path and keep their normal explosion-resistant behavior.
This mod does not change how crates are sealed, placed, broken, or crafted.<br>
For Immersive Engineering's own crate behavior, see <link;storage_crates;Storage Crates>.<np>
<&crate_icons>Compatible crate-like items can opt in through the wimc:industrial_crates item tag.
Only inventory preview detection is added; tagged items should already store their contents on the item stack through modern components, block entity data, or an item capability.
